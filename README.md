# GateKeeper Website

Multi-page marketing site for the GateKeeper Windows security suite.

## Pages

- Home
- Features
- Pricing
- Contact

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Run production locally

```bash
npm run start
```

## Key files

- app/root.tsx (shared layout and navigation)
- app/routes/home.tsx (home page)
- app/routes/features.tsx (features page)
- app/routes/pricing.tsx (pricing page)
- app/routes/contact.tsx (contact page)
- app/app.css (theme and animations)
